Primera semana

Desarrollada del, 21 de Agosto, al 25 de Agosto

¿Que temas aprendió?

Durante la primera semana del curso, se pudo aprender sobre las bases fundamentales para el aprendizaje de tecnología web, esto incluyó la presentación del sílabo del curso, el cual proporcionó una descripción detallada del contenido que se abordaría a lo largo del semestre. El sílabo permitió comprender los objetivos de aprendizaje que se esperan lograr, es decir, las habilidades y conocimientos que deberíamos adquirir al final del curso, también se presentaron los diferentes temas que se tratarían, lo cual brindó una visión general de los conceptos y áreas clave que se exploraran durante la duración de este semestre académico.

Definiciones

Estrategias de aprendizaje y tecnologías de desarrollo

Tener una estrategia de aprendizaje efectiva permite maximizar el proceso de adquisición de conocimientos y habilidades.

Como funciona la web y que aplicaciones de internet existen

Se basa en una serie de conceptos fundamentales, entre ellos el Sistema de Nombres de Dominio (DNS) y el protocolo TCP/IP.

Tecnologías web

Se refiere a las tecnologías web fundamentales, como HTML, CSS y JavaScript, así como el uso de web APIs para la creación de aplicaciones web.

Diseño de experiencia dee interfaz

UX se enfoca en comprender las necesidades y expectativas de los usuarios, UI se centra en la apariencia visual de una aplicación.

Procedimientos

  1. Identificar el número IP de nuestra PC: Se menciona el procedimiento de utilizar el comando "ipconfig" en la línea de comandos para obtener la dirección IP de nuestro equipo.
  2. Comprobar la conectividad del servidor: Se menciona el procedimiento a utilizar con el comando "ping" seguido de la dirección IP del servidor para verificar la conectividad.
  3. Verificar el recorrido de servidores: Se menciona el procedimiento de utilizar el comando "tracert" seguido de la dirección de un sitio web para ver el recorrido de los servidores hasta llegar a ese sitio.

Resultados

  1. Se destacó la importancia de tener una estrategia de aprendizaje efectiva para maximizar el proceso de adquisición de conocimientos y habilidades.
  2. Se presentaron diferentes tecnologías utilizadas en el desarrollo de software, tanto en el ámbito web como en el desarrollo de aplicaciones nativas.
  3. Se explicó el funcionamiento básico de la web, incluyendo conceptos como DNS y TCP/IP.
  4. Se analizó el contexto de las aplicaciones de internet y se presentaron los diferentes roles involucrados en su desarrollo, como el desarrollador frontend y backend.
