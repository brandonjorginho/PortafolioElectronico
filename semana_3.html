Tercera semana

Desarrollada del, 04 de Setiembre, al 08 de Setiembre

¿Que temas aprendió?

Durante esta semana, se adquirieron conocimientos sobre varios temas relacionados con JavaScript y su aplicación en el desarrollo de páginas web interactivas y dinámicas. Se aprendió sobre este lenguaje de programación y su papel fundamental en la creación de interactividad y dinamismo en las páginas web, JavaScript permite la implementación de funcionalidades como validaciones, animaciones y manipulación del contenido en tiempo real. Se exploraron los eventos en JavaScript y cómo se pueden utilizar para responder a las acciones del usuario, como hacer clic en un botón o mover el mouse. Los eventos permiten la ejecución de código en respuesta a la interacción del usuario, lo que mejora la experiencia de usuario y la interactividad de la página. Se profundizó en la manipulación del DOM (Document Object Model) utilizando JavaScript. El DOM es una representación estructurada del contenido de una página web, y JavaScript ofrece poderosas herramientas para acceder y modificar dinámicamente el contenido, la estructura y los estilos del DOM. Esto permite crear efectos visuales, actualizar datos en tiempo real y mejorar la usabilidad de la página. Además, se aprendió sobre la validación de formularios en JavaScript. Se exploraron las funciones de validación que permiten verificar y asegurar que los datos ingresados por el usuario en los formularios cumplan con ciertos criterios, como campos obligatorios, formatos de fecha o contraseñas seguras. La validación de formularios es una práctica importante para garantizar la integridad de los datos y brindar una experiencia de usuario fluida.

Definiciones

JavaScript

s un lenguaje de programación utilizado en el desarrollo web para crear interactividad y dinamismo en las páginas.

Eventos

Son acciones o sucesos que ocurren en una página web, como hacer clic en un elemento, mover el mouse o presionar una tecla.

Manipulación del DOM

Es una representación estructurada del contenido de una página web.

Validación de formularios

Se refiere a la verificación de los datos ingresados por el usuario en los campos de un formulario.

Procedimientos

  1. Renderizado: Se llevó a cabo el renderizado, que consiste en representar y pintar los píxeles en la pantalla a partir del árbol de renderizado final, este proceso crítico fue realizado por el motor de renderizado del navegador, siguiendo una ruta específica.
  2. JavaScript: se brindó especial atención al lenguaje de programación JavaScript y su papel fundamental en la creación de interactividad y dinamismo en las páginas web.
  3. Eventos: Se exploraron los eventos en JavaScript, los cuales permiten responder a las acciones del usuario, como hacer clic en un botón o mover el mouse.
  4. Manipulación del DOM con JavaScript: se adquirieron conocimientos sobre la manipulación del DOM con JavaScript, lo que posibilita cambiar dinámicamente el contenido, la estructura y los estilos de una página web.
  5. Validación de formularios: Por último, se abordó la validación de formularios en JavaScript, aprendiendo a utilizar las funciones de validación para asegurar que los datos ingresados por el usuario cumplan con los criterios requeridos.

Resultados

  1. Se creó el Document Object Model (DOM) a partir de la respuesta HTML del servidor, esto permitió representar la estructura y contenido de la página web.
  2. Se creó el CSS Object Model (CSSOM) que representa los estilos CSS utilizados en la página, esto permitió aplicar los estilos correspondientes a cada elemento del DOM.
  3. Se realizó la combinación del DOM y el CSSOM en el Render Tree, que representa los elementos visibles en la página y sus estilos aplicados, esto permitió determinar qué elementos se mostrarán en la pantalla y cómo se verán.
  4. Se realizo el cálculo del layout, donde se determinó la posición y el tamaño exactos de cada elemento en la página, esto implicó establecer las dimensiones y la ubicación de los elementos en relación con otros elementos y con el contenedor principal.
