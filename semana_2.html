Segunda semana

Desarrollada del, 28 de Agosto, al 01 de Setiembre

¿Que temas aprendió?

Durante esta semana, se adquirieron conocimientos sobre la importancia de la plataforma web abierta como estándar para el desarrollo de aplicaciones web, la estructura y contenido de las páginas web utilizando HTML, el diseño y presentación de estas utilizando CSS, también se realizó una introducción al diseño de una página y su posicionamiento utilizando técnicas como Flexbox y Grid Layout, estos conocimientos nos proporcionaron una base sólida para la creación y diseño de aplicaciones web.

Definiciones

Open Web Platform

Es una colección de tecnologías y estándares desarrollados por organismos como el W3C, Unicode Consortium.

HTML (HyperText Markup Language)

Es un lenguaje de marcado que define la estructura y semántica del contenido de las páginas web..

CSS (Cascading Style Sheets)

Es un lenguaje de estilo utilizado para definir la presentación y el diseño de las páginas web.

Unidades de medida en CSS

Se utilizan para definir la altura, anchura y margen de los elementos en CSS.

Procedimientos

  1. Creación del Document Object Model (DOM): Se creó el DOM a partir de la respuesta HTML del servidor, esto implico convertir el código HTML en una estructura de árbol que representa la jerarquía de elementos de la página.
  2. Creación del CSS Object Model (CSSOM): Se creó el CSSOM, que es una representación del CSS utilizado en la página, esto implico analizar y procesar las reglas CSS para determinar los estilos aplicados a cada elemento del DOM.
  3. Combinación del DOM y el CSSOM en el Render Tree: Se combino el DOM y el CSSOM para crear el Render Tree, que es una representación de los elementos visibles en la página y sus estilos aplicados, esto implico determinar qué elementos se mostrarán en la pantalla y cómo se verán.
  4. Cálculo del layout: Una vez que se tuvo el Render Tree, se calcula el layout, es decir, se determina la posición y el tamaño exactos de cada elemento en la página, esto implico establecer las dimensiones y la ubicación de los elementos en relación con otros elementos y con el contenedor principal.

Resultados

  1. Se creó el DOM a partir de la respuesta HTML del servidor, lo que permitió representar la estructura y contenido de la página web.
  2. Se creo el CSSOM, que represento los estilos CSS utilizados en la página, esto permitió aplicar los estilos correspondientes a cada elemento del DOM.
  3. Se combino el DOM y el CSSOM para crear el Render Tree, que representan los elementos visibles en la página y sus estilos aplicados, con esto se determino qué elementos se mostrarán en la pantalla y cómo se verán.
