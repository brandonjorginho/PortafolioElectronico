Cuarta semana

Desarrollada del, 11 de Setiembre, al 15 de Setiembre

¿Que temas aprendió?

Al estudiar la historia y características de JavaScript, se aprendió sobre su origen en los años 90 y su propósito de brindar interactividad a las páginas web, también se comprende cómo se incluye en el código HTML mediante etiquetas de script, además, se adquierió conocimiento sobre los comentarios en JavaScript y PHP, así como la forma de declarar variables y los diferentes tipos de datos que se pueden utilizar en JavaScript. Se exploró el uso de operadores para realizar diversas operaciones y se estudian las estructuras de control de flujo para controlar el flujo de ejecución del programa. Se adquirió comprensión sobre la creación de funciones y la forma en que se pueden utilizar eventos para responder a acciones del usuario, también se aprende sobre objetos y arrays en JavaScript, y se obtiene una idea de las diversas aplicaciones que se pueden crear utilizando este lenguaje, como páginas web interactivas, aplicaciones web, aplicaciones móviles híbridas y servidores de aplicaciones.

Definiciones

JavaScript

Es un lenguaje de programación interpretado, orientado a objetos y basado en prototipos, es utilizado principalmente para crear aplicaciones web interactivas y dinámicas.

ECMAScript

Es el estándar en el que se basa JavaScript, desde 2012, todos los navegadores modernos soportan completamente ECMAScript. La sexta versión de ECMAScript, fue publicada en 2015.

Funciones en JavaScript

Son bloques de código diseñados para realizar una tarea en particular y se ejecutan cuando se invocan. Los argumentos se pasan por valor y los objetos se pasan por referencia.

Objetos en JavaScript

Son colecciones de propiedades, estas pueden ser métodos o valores. Los objetos pueden ser creados utilizando la palabra clave "new" seguida del nombre del objeto.

Procedimientos

  1. Se describió JavaScript como un lenguaje de programación interpretado y orientado a objetos utilizado en el desarrollo web, enfatizando su capacidad para agregar interactividad a las páginas.
  2. Se explicó la inclusión de JavaScript en HTML utilizando etiquetas de script, lo que permite que el código JavaScript se ejecute en el navegador.
  3. Se mencionaron los comentarios en JavaScript y su importancia para agregar notas y hacer el código más legible. Se proporcionó información sobre los comentarios de una sola línea y de varias líneas.
  4. Se abordaron los conceptos de variables y tipos de datos en JavaScript, destacando la declaración de variables con "var" o "let" y la diversidad de tipos de datos compatibles.
  5. Se describieron los operadores en JavaScript, incluyendo operaciones matemáticas, comparaciones y asignaciones.
  6. Se explicaron las estructuras de control de flujo en JavaScript, como las sentencias if-else, while, for y switch, y se proporcionó una visión general de cómo se utilizan para tomar decisiones y repetir acciones.
  7. Se analizaron las funciones en JavaScript, que permiten encapsular bloques de código reutilizables, y se destacó su capacidad para recibir parámetros y devolver valores.
  8. Se mencionó la importancia de los eventos en JavaScript, que permiten responder a acciones del usuario o cambios en el estado de la página.
  9. Se describieron los objetos en JavaScript como colecciones de propiedades y métodos, y se mencionaron las diferentes formas de crear objetos.
  10. Se explicaron los arrays en JavaScript como estructuras de datos que almacenan múltiples valores en una sola variable.
  11. Finalmente, se mencionaron las diversas aplicaciones que se pueden crear con JavaScript, como páginas web interactivas, aplicaciones web, aplicaciones móviles híbridas y servidores de aplicaciones.

Resultados

  1. Interactividad mejorada: Al utilizar JavaScript en el desarrollo web, se pudo crear páginas y aplicaciones más interactivas, por ejemplo, se pudo agregar eventos a elementos HTML para responder a acciones del usuario, como hacer clic en un botón o mover el mouse, esto permite una experiencia más dinámica y atractiva para los usuarios.
  2. Validación de formularios en tiempo real: Con JavaScript, se pudo realizar validaciones de formularios en el lado del cliente antes de enviar los datos al servidor, esto para evitar que los usuarios envíen información incorrecta o incompleta y brinda una experiencia más fluida.
  3. Manipulación del DOM: JavaScript permite manipular el DOM de una página web, lo que significa que se puede modificar y actualizar elementos HTML y CSS en tiempo real, por ejemplo, se puedo cambiar el contenido de un elemento, agregar o eliminar elementos, cambiar estilos o animar elementos para crear efectos visuales atractivos.
  4. Implementación de funcionalidades adicionales: JavaScript nos brinda la capacidad de agregar funcionalidades personalizadas a tus aplicaciones web, por ejemplo, se pudo crear cálculos matemáticos, implementar algoritmos y realizar operaciones.
