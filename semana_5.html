Quinta semana

Desarrollada del, 18 de Setiembre, al 22 de Setiembre21

¿Que temas aprendió?

Se aprendió cómo utilizar el elemento canvas de HTML5 junto con JavaScript para dibujar gráficos y animaciones en una página web, se proporcionaron ejemplos de código para dibujar formas básicas como líneas, rectángulos y círculos, así como para crear animaciones mediante el uso del método setInterval(). Además, se discutió cómo utilizar el canvas para crear juegos simples.

Definiciones

Canvas

Eun elemento HTML5 que permite dibujar gráficos y animaciones en una página web utilizando JavaScript.

Contexto de dibujo

Es el objeto que se utiliza para dibujar, se obtiene mediante el método getContext() del elemento canvas.

Coordenadas

El punto (0,0) se encuentra en la esquina superior izquierda y los valores positivos de x y y se extienden hacia la derecha y hacia abajo, respectivamente.

Animación

Se puede crear una animación en el canvas utilizando el método setInterval() de JavaScript para actualizar el dibujo en el canvas en intervalos regulares.

Procedimientos

  1. Obtener el contexto de dibujo: Para dibujar en canvas, primero tuvimos que obtener el contexto de dibujo utilizando el método getContext() del elemento canvas, el contexto de dibujo es el objeto que se utiliza para dibujar en el canvas.
  2. Dibujar formas básicas: Canvas nos proporciona métodos para dibujar formas básicas como líneas, rectángulos, círculos y arcos. Estos métodos incluyen strokeRect(), fillRect(), arc(), lineTo(), entre otros.
  3. Dibujar texto: Canvas también nos permitió dibujar texto utilizando el método fillText() o strokeText(), se puede especificar la fuente, el tamaño y el color del texto.
  4. Dibujar imágenes: Es posible dibujar imágenes en canvas utilizando el método drawImage(), además se puede especificar la imagen a dibujar, la posición y el tamaño.
  5. Animación: Se puede crear animaciones en el canvas utilizando el método setInterval() de JavaScript para actualizar el dibujo en el canvas en intervalos regulares, esto nos permitió crear animaciones suaves y fluidas.
  6. Transformaciones: Canvas nos permitió aplicar transformaciones a los dibujos, como rotaciones, escalas y traslaciones, con ello creamos dibujos más complejos y dinámicos.

Resultados

  1. Al obtener el contexto de dibujo y utilizar el método arc() de canvas, se pudo dibujar un círculo.
  2. Utilizando el contexto de dibujo y los métodos rect() o fillRect() de canvas, se logró dibujar un cuadrado.
  3. Aplicando el método setInterval() y actualizando la posición del círculo en cada intervalo, se logró una animación de movimiento.
  4. Utilizando los métodos de transformación del contexto de dibujo, como rotate() o translate(), se aplicó una transformación al cuadrado.
