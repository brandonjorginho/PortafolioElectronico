Desarrollada del, 25 de Setiembre, al 29 de Setiembre21

¿Que temas aprendió?

En el transcurso de la sexta semana, se abordo el tema de React y su funcionamiento, React es una biblioteca de JavaScript ampliamente utilizada para construir interfaces de usuario interactivas y escalables, durante esta semana, se ha explorado los conceptos fundamentales de React, desde la instalación inicial hasta la comprensión de las estructuras de directorio utilizadas en el desarrollo de aplicaciones React. A lo largo de este tema, también aprendimos cómo instalar Node.js, un entorno de ejecución esencial para trabajar con React, Node.js permite ejecutar código JavaScript fuera del navegador y proporciona un conjunto de herramientas y bibliotecas para el desarrollo web moderno.

Definiciones

React

Es una biblioteca de JavaScript que se utiliza para construir interfaces de usuario interactivas y reactivas.

Componente

Es una unidad independiente y reutilizable de la interfaz de usuario que encapsula la lógica y la presentación de una parte de la aplicación.

Node.js

Es un entorno de ejecución de JavaScript basado en el motor V8 de Google Chrome, permite ejecutar código JavaScript fuera del navegador

Estado

Es un objeto que representa la información relevante para un componente en un momento dado.

Procedimientos

  1. Instalación de Node.js: Este procedimiento implica descargar e instalar Node.js en el sistema, este se puede obtener desde el sitio web oficial de Node.js (https://nodejs.org) y está disponible para diferentes sistemas operativos, la instalación de Node.js proporcionará el entorno de ejecución necesario para ejecutar código JavaScript fuera del navegador.
  2. Inclusión de React y ReactDOM a través de un CDN: En lugar de descargar e incluir localmente las librerías de React y ReactDOM, se ha optado por utilizar un CDN (Content Delivery Network), un CDN es un servicio que almacena y distribuye archivos estáticos, como librerías de JavaScript, en servidores ubicados en diferentes ubicaciones geográficas para mejorar la velocidad de entrega.

Resultados

  1. Configuración del entorno de desarrollo: Al instalar Node.js, se estableció un entorno de ejecución de JavaScript que nos permitirá utilizar React, tanto en el lado del servidor como en el cliente.
  2. Acceso a las bibliotecas de React: Al incluir las librerías React y ReactDOM a través de un CDN, tendremos acceso inmediato a las funciones y capacidades de React.
  3. Ahorro de tiempo en la configuración inicial: Al crear un proyecto en React, se estableció una estructura de directorios y se generaron los archivos de configuración necesarios.
  4. Facilidad de desarrollo y mantenimiento: Al utilizar React en conjunto con Node.js y las librerías React y ReactDOM, tendremos acceso a un ecosistema de herramientas y bibliotecas que nos facilitarán el desarrollo y el mantenimiento de una aplicación.
